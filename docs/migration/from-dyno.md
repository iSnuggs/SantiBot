# Migrating from Dyno

Switching from Dyno to SantiBot gives you all the same moderation and management features plus music, economy, games, and full customization — completely free.

## Feature Mapping

| Dyno Feature | SantiBot Equivalent | Command |
|-------------|---------------------|---------|
| Automod | Anti-Spam/Raid/Alt | `.antispam`, `.antiraid`, `.antialt` |
| Mod Log | Server Logging | `.log` (20 event types) |
| Auto Roles | Auto Assign Roles | `.autoassignrole` |
| Reaction Roles | Reaction/Button Roles | `.reroadd`, `.btr` |
| Custom Commands | Expressions | `.exadd` |
| Announcements | Greetings | `.greet`, `.bye` |
| Starboard | Starboard | `.starboardchannel` |
| Giveaways | Giveaways | `.ga start` |
| AFK | AFK | `.afk` |
| Reminders | Reminders | `.remind`, `.remindrecurring` |
| Suggestions | Suggestions | `.suggest` |
| Auto Message | Repeaters | `.repeat` |
| Slowmode | Channel Moderation | Built into Discord + `.imgonly`, `.linkonly` |
| Twitch Alerts | Stream Notifications | `.stream twitch` |
| YouTube Alerts | Feed/Upload Notifs | `.ytuploadnotif` |
| Tags | Quotes | `.quoteadd`, `.quoteshow` |

## What SantiBot Adds (Not in Dyno)

- Full music system (YouTube, SoundCloud, 1000+ sources)
- Economy system with currency, gambling, shops
- XP & Leveling with customizable rank cards
- Games (trivia, hangman, fishing, typing contests)
- Waifu collecting system
- Forms (DM-based questionnaires)
- Auto Purge (scheduled message cleanup)
- Polls with button-based voting
- Plugin system (Medusa) for custom extensions
- Self-hostable — your data, your server
- Web dashboard for configuration
- Completely free, no premium tiers

## Setup

1. Follow the [Getting Started](../getting-started.md) guide
2. Configure your features through commands or the dashboard
3. Dyno command aliases are supported where possible for familiar usage
