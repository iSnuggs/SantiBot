# Features Overview

SantiBot combines the best features of NadekoBot and Dyno into one self-hostable bot.

## Moderation & Server Management

| Feature | Commands | Description |
|---------|----------|-------------|
| Anti-Spam | `.antispam` | Automatic spam detection and action |
| Anti-Raid | `.antiraid` | Protect against mass joins |
| Anti-Alt | `.antialt` | Detect and handle alt accounts |
| Warnings | `.warn`, `.warnlog` | Issue warnings with auto-punishments |
| Muting | `.mute`, `.unmute` | Timed mutes (chat/voice) |
| Pruning | `.prune` | Bulk message deletion with filters |
| Auto Purge | `.autopurgeset` | Scheduled automatic message cleanup |
| Logging | `.log` | 20 event types with per-channel routing |
| Honeypot | `.honeypot` | Bait channels for bad actors |

## Roles
