    NadekoBot Documentation¶

    Inviting Nadeko¶

    There are two versions of Nadeko, a public bot and a self-hostable bot.

    To invite public Nadeko to your server or to view its commands, click on the buttons below:

    Add Nadeko to your server View commands

    To self-host your own Nadeko, use the guides below:

    • Desktop guide (Windows/Linux/macOS)
    • CLI guide (Linux/macOS)
    • Docker guide
    • From source guide

    In case you need any help, join our Discord server where we may provide support.

    About Nadeko¶

    NadekoBot is an open source project. Any issues with the bot may be filed here.

    If you're unsure whether something is an issue, ask in our support server first.

    Donations are welcome, and we rely on your contributions to help keep the project alive.
